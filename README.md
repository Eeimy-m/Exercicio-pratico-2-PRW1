# Exercicio-pratico-2-PRW1
Desenvolva o frontend de uma aplicação que simula uma clínica médica. Neste sistema, o usuário pode realizar as seguintes operações:

Cadastrar, listar, atualizar e remover médicos.
Cadastrar, listar, atualizar e remover pacientes.
Cadastrar e remover consultas.
A aplicação a ser desenvolvida deve utilizar como backend de desenvolvimento o serviço disponível em https://ifsp.ddns.net/webservices/clinicaMedica/. Note que o frontend deve ser flexível e sempre usar os dados provenientes do backend (ou seja, jamais assuma que um valor de ID = X sempre existirá ou será válido!)

Além das operações CRUD mencionadas acima, o sistema deverá suportar três operações extras:
O usuário pode selecionar uma especialidade e, com isso, ver todos os médicos daquela especialidade.
O usuário pode selecionar um médico e ver todas as consultas daquele médico.
O usuário pode selecionar um paciente e ver todas as consultas daquele paciente.
Um exemplo da aplicação em execução, mas onde nem todos os recursos estão implementados, está disponível aqui: https://ifsp.ddns.net/angular/clinicaMedica/
